Computer Security: Spoofing Attacks

This has to be one of the coolest projects I have done and it was all about spoofing. I explored multiple attacks that used spoofing through VirtualBox, Kali Linux, and Wireshark. I practiced MAC address spoofing to mimic identity changes within a network, created a cloned website through NGINX to mimic web spoofing, and altered the DNS settings to mimic the redirecting nature of cache poisoning. This project gave me hands-on, real-world practice working with cyber attacks as well as countermeasures.

Advanced Networking: VLAN and Inter-VLAN Routing

In this project, I set up VLAN's on multiple switches to split a larger network into smaller parts. I also configured inter-VLAN routing on a router so the different departments could still communicate safely. This lab helped me understand advanced switching, how network separation works, and how it improves both performance and security in business networks.
